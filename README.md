# DesignX

面向系统设计人员与模块负责人的本地研发治理工作台。当前版本覆盖代码仓、知识库、分析任务与发现治理四条核心工作流，并使用浏览器本地存储保留演示数据与操作状态。

## 本地运行

```bash
npm install
npm run dev
```

浏览器开发地址默认为 `http://localhost:5173`。

启动 Electron 桌面壳：

```bash
npm run dev:electron
```

执行类型检查与生产构建：

```bash
npm run build
```

## 交互范围

- 添加、同步代码仓并查看索引状态
- 导入知识文档、生成草稿、校验来源并发布不可变版本
- 创建分析任务，观察本地模拟的阶段进度并重试失败任务
- 按仓库、任务、类型、等级和状态筛选发现
- 对照代码证据与知识依据，确认并忽略发现
- 配置模型连接并执行本地连通性反馈

所有演示状态均保存在当前设备的 `localStorage` 中。Electron 渲染进程启用了上下文隔离，未开放 Node.js 或文件系统能力。
